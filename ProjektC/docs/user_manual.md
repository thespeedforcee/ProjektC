# User Manual

## Launch and Controls
1. Launch the game executable. You will be greeted by the start screen.
2. Press the **S** key to start the game.
3. Control the snake using the **Arrow Keys** (Up, Down, Left, Right). 
4. Eat red apples to score points (10 points each). As you score points, the snake accelerates, dynamically increasing the difficulty level.

## Game Rules
* Hitting the screen boundaries (the brown wall) or your own tail ends the game (Game Over).
* On the game over screen, press **R** to restart the game and return to the main menu, or press **ESC** to exit the application.
* Your high score is automatically saved to the disk and loaded upon the next launch.
